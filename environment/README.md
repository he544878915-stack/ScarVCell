# Computational environment

The released scripts use Python and R/Bioconductor packages listed in this directory. MAGMA 1.10 and PLINK 2.00a7.1 are external command-line dependencies for the ancestry-aware genetic analysis. Public source archives are not redistributed; accession identifiers, released input types, hashes and analysis units are listed in `config/dataset_screening_and_technical_audit.csv` and `config/public_input_provenance.csv`.

Exact paths are intentionally not fixed. Set the project root and public-data staging directories for the local environment before execution. Dataset-level effects remain primary and biological units must not be replaced by cells or technical libraries.
